## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Abdo1454/clinic-app-react.git
```

Move into the project folder:

```bash
cd clinic-app-react
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

## 🏗️ Build

Create a production build:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
