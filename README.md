# 🏥 MediCore

A modern and responsive healthcare landing page built with React and Vite. The project focuses on providing a clean user experience, responsive layouts, and reusable React components.

## 🚀 Live Features

* Responsive Navigation Bar
* Mobile Hamburger Menu
* Hero Section
* Services Section
* Smooth CSS Animations
* Modern UI Design
* Mobile-First Responsive Layout
* Reusable React Components

## 🛠️ Tech Stack

* React
* JavaScript (ES6+)
* Vite
* CSS3
* Flexbox
* Git & GitHub

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Abdo1454/clinic-app-react.git
```

Move into the project folder:

```bash
cd clinic-app-react
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

## 🏗️ Build

Create a production build:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
